# go

# 環境構築

## ダウンロード

* https://go.dev/dl/
    * go1.25.5.linux-amd64.tar.gz などをダウンロード
* 展開
    * tar xvzf go1.25.5.linux-amd64.tar.gz
* パス
    * 展開したところにパスを通す

.bashrc に以下を追記して .bashrc を読み直す。（bash 再起動)
```
export PATH=$PATH:/home/ishii/ishii/ollama/go/bin
export GOPATH=/home/ishii/ishii/ollama/go/gopath
```

# 新規project作成

```
mkdir myproject
cd myproject
go mod init myproject
```
